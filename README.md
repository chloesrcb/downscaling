# Downscaling

This repository contains code and experiments for statistical downscaling of precipitation data between two datasets with different spatial and temporal resolutions:

- **OMSEV**: an in situ observational dataset, sub-kilometric and 5-minute records
- **COMEPHORE**: a gauge-radar precipitation reanalysis product, 1 km by 1 km pixel grid and hourly records

The main objective is to learn a relationship between the large-scale precipitation information 
provided by COMEPHORE and the local in situ precipitation observations from OMSEV.

More specifically, this project focuses on the statistical modeling of precipitation, with particular attention to extremes.
